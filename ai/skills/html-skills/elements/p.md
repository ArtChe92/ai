# p

## Сценарии использования

Абзац текста.

**Когда выбрать другой элемент:**

- Заголовок → `h1`-`h6`
- Цитата → `blockquote`
- Подпись к изображению → `figcaption`
- Контактная информация → `address`
- Просто обёртка для стилей → `div` или `span`

## Связанные элементы

| Элемент  | Связь                      |
| -------- | -------------------------- |
| `strong` | Дочерний — важный текст    |
| `em`     | Дочерний — акцент          |
| `a`      | Дочерний — ссылки в тексте |
| `br`     | Дочерний — перенос строки  |

> См. strong.md, a.md

## Необходимые атрибуты

Специфичных атрибутов нет.

## Визуальные состояния

Не применимо.

## Правила доступности

- Не использовать пустые `<p></p>` для отступов — использовать CSS
- Не вкладывать блочные элементы (`div`, `ul`, `table`) внутрь `p`
- Длинные абзацы разбивать для читаемости

## SEO

- Основной текстовый контент для индексации
- Ключевые слова в первых абзацах важнее

## Пример

```tsx
<article>
    <h1>Заголовок статьи</h1>

    <p>
        Первый абзац с <strong>важным текстом</strong> и<a href="/link">ссылкой</a> внутри.
    </p>

    <p>Второй абзац продолжает мысль. Каждый абзац — логически завершённая мысль.</p>
</article>
```
