# time

## Сценарии использования

Дата, время или временной промежуток в машиночитаемом формате.

**Когда выбрать другой элемент:**

- Просто текст без машиночитаемой даты → `span`
- Нет конкретной даты ("скоро", "недавно") → `span`

**Примеры использования:**

- Дата публикации статьи
- Время события
- Дата рождения
- Длительность (duration)
- Дедлайны

## Связанные элементы

| Элемент   | Связь                      |
| --------- | -------------------------- |
| `article` | Родитель — дата публикации |
| `footer`  | Родитель — мета-информация |
| `p`       | Родитель — в тексте        |

> См. article.md

## Необходимые атрибуты

### По ситуации

| Атрибут    | Когда                                        |
| ---------- | -------------------------------------------- |
| `datetime` | Всегда, если текст отличается от ISO-формата |

### Форматы datetime

| Тип          | Формат                   | Пример                   |
| ------------ | ------------------------ | ------------------------ |
| Дата         | `YYYY-MM-DD`             | `2026-01-27`             |
| Время        | `HH:MM` или `HH:MM:SS`   | `14:30`                  |
| Дата и время | `YYYY-MM-DDTHH:MM`       | `2026-01-27T14:30`       |
| С таймзоной  | `YYYY-MM-DDTHH:MM±HH:MM` | `2026-01-27T14:30+03:00` |
| Год и месяц  | `YYYY-MM`                | `2026-01`                |
| Год          | `YYYY`                   | `2026`                   |
| Неделя       | `YYYY-Www`               | `2026-W04`               |
| Длительность | `PnYnMnDTnHnMnS`         | `PT2H30M` (2ч 30мин)     |

⚠️ Если текст внутри `time` уже в ISO-формате, `datetime` не обязателен.

## Визуальные состояния

Не применимо.

## Правила доступности

- Screen reader читает текст внутри, не `datetime`
- Текст должен быть понятен пользователю
- `datetime` — для машин (поисковики, календари)

## SEO

- Поисковики используют `datetime` для понимания актуальности
- Важно для новостей, событий, статей
- Улучшает rich snippets

## Пример

```tsx
{
    /* Дата публикации */
}
<article>
    <h1>Новые возможности React 19</h1>
    <p>
        Опубликовано: <time dateTime="2026-01-27">27 января 2026</time>
    </p>
</article>;

{
    /* Дата и время события */
}
<p>
    Вебинар состоится <time dateTime="2026-02-15T19:00+03:00">15 февраля в 19:00 МСК</time>
</p>;

{
    /* Длительность */
}
<p>
    Продолжительность курса: <time dateTime="PT8H">8 часов</time>
</p>;

{
    /* В футере статьи */
}
<footer>
    <time dateTime="2026-01-27">27.01.2026</time>
    <address>
        Автор: <a href="/authors/1">Имя автора</a>
    </address>
</footer>;
```
