# A11y Review

Проведи accessibility-ревью указанных файлов.

## Инструкции

1. **Загрузи и используй скилл** — html-skills

2. **Определи используемые элементы** — проанализируй файлы

3. **Загрузи правила для каждого элемента** — для каждого найденного элемента компонента прочитай соответствующий файл из `elements`

4. **Проверь каждый элемент** по правилам из скилла

5. **Сформируй отчёт** в формате чек-листа
