# form

## Сценарии использования

Контейнер для группы полей ввода с отправкой данных.

**Когда выбрать другой элемент:**

- Фильтры без отправки на сервер → `div` с обработчиками
- Одиночное поле поиска → можно без `form`, но с `form` лучше (Enter работает)

## Связанные элементы

| Элемент    | Связь                        |
| ---------- | ---------------------------- |
| `input`    | Поля ввода                   |
| `textarea` | Многострочный ввод           |
| `select`   | Выпадающие списки            |
| `button`   | Кнопки отправки / сброса     |
| `fieldset` | Группировка связанных полей  |
| `label`    | Подписи полей                |
| `search`   | Родитель — если форма поиска |

> См. input.md, button.md, search.md

## Необходимые атрибуты

### По ситуации

| Атрибут           | Когда                                        |
| ----------------- | -------------------------------------------- |
| `action`          | URL для отправки (если не SPA)               |
| `method`          | GET / POST (если не SPA)                     |
| `novalidate`      | Отключить браузерную валидацию (своя логика) |
| `enctype`         | `multipart/form-data` для загрузки файлов    |
| `aria-label`      | Если нет видимого заголовка формы            |
| `aria-labelledby` | Связь с заголовком формы                     |

## Визуальные состояния

| Состояние | Описание                              |
| --------- | ------------------------------------- |
| `default` | Базовый вид                           |
| `loading` | Форма отправляется — блокировка полей |
| `error`   | Есть ошибки валидации                 |
| `success` | Успешная отправка (опционально)       |

## Правила доступности

- Группировать связанные поля в `fieldset` с `legend`
- Форма должна иметь понятное название (`aria-label` или заголовок)
- Ошибки валидации — показывать сразу у полей, не только в общем блоке
- После отправки — фокус на результат или первую ошибку

## SEO

Не влияет напрямую.

## Пример

```tsx
<form onSubmit={handleSubmit} noValidate aria-label="Форма регистрации">
    <fieldset disabled={isLoading}>
        <legend>Личные данные</legend>

        <div className="field">
            <label htmlFor="name">Имя</label>
            <input id="name" name="name" type="text" required />
        </div>

        <div className="field">
            <label htmlFor="email">Email</label>
            <input id="email" name="email" type="email" required />
        </div>
    </fieldset>

    <button type="submit" disabled={isLoading} aria-busy={isLoading}>
        {isLoading ? 'Отправка...' : 'Зарегистрироваться'}
    </button>
</form>
```
