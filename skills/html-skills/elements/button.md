# button

## Сценарии использования

Интерактивный элемент для выполнения действия.

**Когда выбрать другой элемент:**

- Переход на другую страницу → `a`
- Переход по URL с действием → `a` (даже если выглядит как кнопка)

**Правило:** кнопка выполняет действие, ссылка ведёт куда-то.

## Связанные элементы

| Элемент | Связь                     |
| ------- | ------------------------- |
| `form`  | Родитель для submit/reset |

> См. form.md, a.md

## Необходимые атрибуты

### Всегда

| Атрибут | Назначение                                       |
| ------- | ------------------------------------------------ |
| `type`  | `button` / `submit` / `reset` — всегда указывать |

### По ситуации

| Атрибут         | Когда                              |
| --------------- | ---------------------------------- |
| `disabled`      | Кнопка неактивна                   |
| `aria-label`    | Кнопка без текста (только иконка)  |
| `aria-pressed`  | Toggle-кнопка (вкл/выкл)           |
| `aria-expanded` | Кнопка раскрывает/скрывает контент |
| `aria-busy`     | Кнопка в состоянии загрузки        |

### Типы button

| Тип      | Использование                  |
| -------- | ------------------------------ |
| `button` | Действие без отправки формы    |
| `submit` | Отправка формы (по умолчанию!) |
| `reset`  | Сброс формы                    |

⚠️ Без `type` кнопка в форме = `submit`. Всегда указывай `type="button"` для обычных кнопок.

## Визуальные состояния

| Состояние  | Описание                                             |
| ---------- | ---------------------------------------------------- |
| `default`  | Базовый вид                                          |
| `hover`    | Наведение курсора                                    |
| `focus`    | Фокус с клавиатуры — обязательно видимый             |
| `active`   | Момент нажатия                                       |
| `disabled` | Неактивное — сниженный контраст, cursor: not-allowed |
| `loading`  | Загрузка, например спиннер                           |

## Правила доступности

- Кнопка без текста → обязательно `aria-label`
- Иконка внутри кнопки → `aria-hidden="true"` на иконке
- Состояние загрузки → `aria-busy="true"`
- Toggle-кнопка → `aria-pressed="true/false"`
- Не отключать фокус (`tabindex="-1"`) без причины

## SEO

Не влияет напрямую.

## Пример

```tsx
<button type="button" onClick={handleClick} disabled={isLoading} aria-busy={isLoading}>
    {isLoading ? (
        <Spinner aria-hidden="true" />
    ) : (
        <>
            <SaveIcon aria-hidden="true" />
            Сохранить
        </>
    )}
</button>
```
