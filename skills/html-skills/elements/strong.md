# strong

## Сценарии использования

Важный текст — сильный акцент, влияющий на смысл.

**Когда выбрать другой элемент:**

- Просто визуальное выделение жирным → CSS `font-weight: bold`
- Лёгкий акцент, интонация → `em`
- Название произведения, термин → `cite` или `dfn`

**Правило:** `strong` — для смысловой важности, не для стилей.

## Связанные элементы

| Элемент | Связь                       |
| ------- | --------------------------- |
| `p`     | Родитель — текст абзаца     |
| `em`    | Альтернатива — лёгкий акцент |
| `b`     | Альтернатива — только визуальное выделение |

## Необходимые атрибуты

Специфичных атрибутов нет.

## Визуальные состояния

Не применимо.

## Правила доступности

- Screen reader может выделять голосом или объявлять как "важно"
- Не злоупотреблять — теряется смысл акцента
- Вложенный `strong` усиливает важность

## SEO

- Поисковики учитывают `strong` как сигнал важности
- Не использовать для keyword stuffing

## Пример

```tsx
<p>
    Перед удалением <strong>обязательно сохраните данные</strong>. 
    Это действие нельзя отменить.
</p>

{/* em vs strong */}
<p>
    Я <em>думаю</em>, это хорошая идея. {/* интонация */}
</p>
<p>
    <strong>Внимание:</strong> сервис недоступен. {/* важность */}
</p>
```
