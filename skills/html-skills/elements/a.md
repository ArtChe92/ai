# a (Link)

## Сценарии использования

Переход на другую страницу или URL.

**Когда выбрать другой элемент:**

- Действие без навигации → `button`
- Визуально кнопка, но ведёт на URL → `a` (стилизовать как кнопку)

**Правило:** ссылка ведёт куда-то, кнопка выполняет действие.

## Связанные элементы

| Элемент | Связь                             |
| ------- | --------------------------------- |
| `nav`   | Родитель для навигационных ссылок |

> См. nav.md, button.md

## Необходимые атрибуты

### Всегда

| Атрибут | Назначение                                     |
| ------- | ---------------------------------------------- |
| `href`  | URL перехода — без него ссылка не фокусируется |

### По ситуации

| Атрибут                                         | Когда                             |
| ----------------------------------------------- | --------------------------------- |
| `target="_blank"` + `rel="noopener noreferrer"` | Внешние ссылки                    |
| `aria-label`                                    | Ссылка без текста (только иконка) |
| `download`                                      | Скачивание файла                  |
| `rel="nofollow"`                                | Внешние / рекламные ссылки        |
| `aria-current="page"`                           | Текущая страница в навигации      |

⚠️ Никогда не использовать `href="#"` с `onClick` — это антипаттерн. Если нужно действие → `button`.

## Визуальные состояния

| Состояние | Описание                                 |
| --------- | ---------------------------------------- |
| `default` | Базовый вид                              |
| `hover`   | Наведение — изменение цвета / underline  |
| `focus`   | Фокус с клавиатуры — обязательно видимый |
| `active`  | Момент клика                             |
| `visited` | Посещённая ссылка (если важно для UX)    |

## Правила доступности

- Текст ссылки информативный (не "тут", "подробнее", "ссылка")
- Ссылка без текста (иконка) → обязательно `aria-label`
- Текущая страница в навигации → `aria-current="page"`

## SEO

- Текст ссылки = ключевые слова
- Избегать "кликните сюда", "читать далее"
- Внешние / рекламные ссылки → `rel="nofollow"`
- Внешние ссылки → `rel="noopener noreferrer"`

## Пример

```tsx
<a href="https://docs.example.com" target="_blank" rel="noopener noreferrer">
    <DocsIcon aria-hidden="true" />
    Документация
</a>
```
