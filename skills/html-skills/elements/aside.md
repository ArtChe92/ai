# aside

## Сценарии использования

Дополнительный контент, косвенно связанный с основным.

**Когда выбрать другой элемент:**

- Основной контент страницы → `main`
- Навигация → `nav`
- Самостоятельная статья → `article`
- Просто обёртка для стилей → `div`

**Примеры использования:**

- Боковая панель (sidebar)
- Блок "Читайте также"
- Сноски, цитаты
- Рекламные блоки
- Глоссарий терминов
- Биография автора в статье

**Правило:** если контент можно убрать без потери смысла основного — это `aside`.

## Связанные элементы

| Элемент   | Связь                              |
| --------- | ---------------------------------- |
| `main`    | Сосед — рядом с основным контентом |
| `article` | Родитель или сосед                 |
| `section` | Дочерний — секции внутри aside     |
| `nav`     | Дочерний — навигация в сайдбаре    |

> См. main.md, article.md

## Необходимые атрибуты

### По ситуации

| Атрибут           | Когда                         |
| ----------------- | ----------------------------- |
| `aria-label`      | Несколько `aside` — различать |
| `aria-labelledby` | Есть видимый заголовок        |

## Визуальные состояния

Не применимо.

## Правила доступности

- Screen reader объявляет как "complementary"
- Несколько `aside` → уникальные `aria-label`
- `aside` внутри `article` — относится к статье
- `aside` вне `article` — относится ко всей странице

## SEO

Поисковики понимают, что это дополнительный, а не основной контент.

## Пример

```tsx
{
    /* Боковая панель страницы */
}
<body>
    <header>...</header>

    <main>
        <article>...</article>
    </main>

    <aside aria-label="Боковая панель">
        <nav aria-label="Рубрики">
            <ul role="list">
                <li>
                    <a href="/tech">Технологии</a>
                </li>
                <li>
                    <a href="/design">Дизайн</a>
                </li>
            </ul>
        </nav>

        <section>
            <h2>Популярное</h2>
            <ul>...</ul>
        </section>
    </aside>

    <footer>...</footer>
</body>;

{
    /* Дополнение к статье */
}
<article>
    <h1>Введение в React</h1>
    <p>Основной текст статьи...</p>

    <aside aria-label="Справка">
        <p>
            <strong>JSX</strong> — синтаксическое расширение JavaScript, позволяющее писать
            HTML-подобный код.
        </p>
    </aside>

    <p>Продолжение статьи...</p>
</article>;
```
