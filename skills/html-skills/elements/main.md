# main

## Сценарии использования

Основное содержимое страницы.

**Когда выбрать другой элемент:**

- Боковая панель → `aside`
- Секция контента → `section`
- Самостоятельная статья → `article`

## Связанные элементы

| Элемент   | Связь                    |
| --------- | ------------------------ |
| `header`  | Сосед — шапка страницы   |
| `footer`  | Сосед — подвал страницы  |
| `aside`   | Сосед — боковая панель   |
| `section` | Дочерний — секции внутри |
| `article` | Дочерний — статьи внутри |

> См. header.md, footer.md, aside.md, section.md

## Необходимые атрибуты

### По ситуации

| Атрибут     | Когда                   |
| ----------- | ----------------------- |
| `id="main"` | Для skip-link навигации |

## Визуальные состояния

Не применимо.

## Правила доступности

- Один `main` на страницу
- Не вкладывать в `article`, `aside`, `footer`, `header`, `nav`
- Skip-link в начале страницы → ведёт на `main`

## SEO

Помогает поисковикам определить основной контент страницы.

## Пример

```tsx
<body>
    <a href="#main" className="skip-link">
        Перейти к содержимому
    </a>

    <header>...</header>

    <main id="main">
        <h1>Заголовок страницы</h1>
        <section>...</section>
        <section>...</section>
    </main>

    <footer>...</footer>
</body>
```
